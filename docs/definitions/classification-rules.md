# 分类规则

## 三级纵向结构

1. **一级分类（大类）**：按穿着逻辑划分，如"通裁"、"上装"、"下装"
2. **二级分类（子类）**：具体的服饰品类，例如：衫袄、背心、裙子
3. **三级分类（形制）**：核心物理形制，不可拆分的最小逻辑单位，例如：长衫袄、半臂、圆领袍

## 横向关联与细化

1. **朝代（制式）**：战国、秦汉、魏晋、南北朝、隋唐、五代、宋、明
2. **样式（Style）**："三级类目 + 朝代"，代表特定历史时期的审美标准
3. **款式（Variant）**："样式 + 结构细节（领形/襟型/袖型等）"，对接生产或销售的颗粒度
